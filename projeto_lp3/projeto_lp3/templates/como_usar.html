Como Utilizar

Navegação

Utilize o menu no topo da página para navegar entre as diferentes seções do site:

  • Home: Página inicial com informações gerais.
  • Contato: Formulário para entrar em contato conosco.
  • Produtos: Lista dos nossos produtos.
  • Cadastro de Produtos: Área para cadastrar novos produtos (restrita).

Busca de Produtos

Na página "Produtos", você pode utilizar a barra de busca para encontrar produtos específicos por nome ou descrição.

Cadastro de Produtos

Se você tiver permissão, poderá acessar a área de "Cadastro de Produtos" para adicionar novos produtos ao catálogo. Para isso, preencha o formulário com as informações do produto e clique em "Salvar".

Informações Adicionais

Para mais informações sobre o uso do site, consulte os Termos de Uso e a Política de Privacidade.
